# Client

Next.js / Клиентская часть проекта cowriter

## Getting Started

.env.production

```
NODE_ENV=production
```

.env.development

```
NODE_ENV=development
```

Install packages:

```bash
npm run install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
